COVID-PROJECT
~Abhijeet.D.Bakshi


The project was done to perform basic programming and statistical operation on the COVID-19 data as on 10-09-2020.
The data was taken from the website www.worldometer.com and is open to use.
The file is in .ipynb format.


Instructions:

1) Make sure that you have Anaconda Navigator installed in your pc. or if you are using PyCharm then you will have to convert  the file in .py format.

2) Make sure that you have a internet connection because the Jupyter-Notebook is using plotly library to plot interactive plots.

3) Make sure that you are using the names of the Continent and the Country as specified in the Excel Sheet.
